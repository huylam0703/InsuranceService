# Insurance Service API

## Giới thiệu
Hệ thống quản lý bảo hiểm (Insurance Service) được xây dựng bằng Spring Boot, hỗ trợ:
- Quản lý người dùng
- Quản lý hợp đồng bảo hiểm (Policy)
- Xử lý yêu cầu bồi thường (Claim)

---

## Công nghệ sử dụng
- Java 17+ / 23
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Spring Security
- Swagger (OpenAPI)
